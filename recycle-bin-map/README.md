# YouTube's Recycle Bin - Format Map

> **Note from KVN AUST:** WORKING ON A FIX FOR SORTING BY UPLOAD DATE

This is the community-maintained list of filename format keyphrases that access YouTube's "video graveyard" - the billions of videos with little to no views. Originally compiled by [KVN AUST](https://www.youtube.com/@KVNAUST) with help from the community.

**Source:** [Original Google Doc](https://docs.google.com/document/d/1mV5PhumaIJ8mtH8XmohqXkk5fjK_HlqcineMccPQm5A/)
